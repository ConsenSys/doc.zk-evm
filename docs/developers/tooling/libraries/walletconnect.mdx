Use the WalletConnect protocol to enable users from a multitude of wallets to
log into decentralized applications. Linea is now listed inside chain list for
any project configuration on [WalletConnect Cloud](https://cloud.walletconnect.com).

To learn more about WalletConnect take a look at the official
[page](https://walletconnect.com/).
