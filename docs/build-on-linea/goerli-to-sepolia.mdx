Linea's principal testnet is now Linea Sepolia. We advise you use Linea Sepolia for any testnet 
purposes.

L1 Goerli is in a deprecated state. Finalization and bridging between Linea Goerli (L2) and Goerli 
(L1) will therefore not be available. Linea Goerli will otherwise continue to operate as normal.

We will officially decommission Linea Goerli on May 31 2024.

## Running a Sepolia node

If you run a Linea Goerli testnet node, we recommend you transition to running a Linea Sepolia node 
as soon as possible.

You will need to spin up a new node. The method is the same as before, and as detailed in our [guide](./run-a-node). However, you must use new genesis files specific to Linea Sepolia:

- [Linea Sepolia Geth genesis file](pathname:///files/sepolia/genesis-sepolia-geth.json)
- [Linea Sepolia Besu genesis file](pathname:///files/sepolia/genesis-sepolia-besu.json)

## Updating your dapps

For your dapp to remain functional, you must:

- Deploy any smart contracts anew on Linea Sepolia
- Ensure your dapps reference the new smart contract deployments.

### MetaMask integration

Linea Sepolia has been added as a default testnet in MetaMask in the following releases:

- MetaMask Extension: 11.13.1 — rolled out from March 29 2024.
- MetaMask Mobile: 7.19.0 — rolled out from April 1 2024.

Linea Goerli will be removed from MetaMask Extension v11.17.0 and MetaMask Mobile v7.23, which we 
expect to rollout from May 27 2024.

### Safe

Linea Sepolia is supported at [safe.linea.build](https://safe.linea.build/welcome). 

### Libraries

The following libraries support Linea Sepolia:

- Chainlist
- Viem

The following libraries do not yet support Linea Sepolia:

- WalletConnect
- WETH9

We'll update this list when further updates are available.
